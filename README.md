# CSDS Project

This repository presents a new method for calibrating the CSDS model using data analysis.

The workflow combines:
- data extraction
- data cleaning
- parameter construction
- CSDS model calibration
- convergence filtering
- visualization
- polynomial fitting

The objective is to use data analysis to improve and support the calibration of the CSDS model for rock joint shear behavior.
